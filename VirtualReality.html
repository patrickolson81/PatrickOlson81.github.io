Sanitation Standoff

Challenge

Conceptualize and develop a virtual reality game within a limited timeframe, incorporating Unity XR Core.

Action

For my game, I thought to embrace the chaos of destruction within virtual reality, envisioning a garbage man on strike who's only way to get the union contract is to cause havoc and create filth with the trash he normally collects. I utilized Unity XR Core to incorporate basic virtual reality interactions, incorporated NPC's with pathfinding using Unity NavMesh's and Ragdoll, and added character and depth to the environment with free Unity Store Packages.

Result

The result of my efforts was a demo game that included a start screen, narrative level, and a gameplay level. This demo was able to showcase object interaction, NPC's, 3D Asset integration, and a loose storyline. Although not a complete product, this demo sets a groundwork for future development that will take place.
